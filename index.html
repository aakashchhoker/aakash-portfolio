Portfolio

SuperAdmin (Learning Management System)

Category: Electron base app supported for Mac and window systems

The application is a Learning Management System (LMS) designed to streamline content management. This app enables users to create and edit sessions, courses, tags, programs, and AI voice content efficiently.

Books Delivery App

Category: React Native base app supported for Androids and iPhones

This application is a comprehensive book barcode scanning tool. It allows users to scan barcodes to retrieve customer details, including order history and personal information such as email and phone number. Additionally, users can mark orders as delivered, cancelled, or on hold. If a barcode cannot be scanned, the app also provides a manual search feature for order processing.

Lifely

Category: React Native base app supported for Androids and iPhones

This application offers a weekly meal subscription service designed to alleviate the stress of dinner preparations. It helps users avoid the last-minute rush to the grocery store, the hectic kitchen environment, and the risk of cooking mishaps.
